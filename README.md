# Portafolio de Evidencias – HTML/CSS/JS

Este proyecto reúne en un solo repositorio todas las actividades prácticas desarrolladas durante el curso. Su objetivo es centralizar los trabajos, aplicar un diseño unificado y permitir una navegación fácil entre cada entrega.

## Objetivo del Proyecto

Crear un portafolio completo y funcional, donde cada actividad esté integrada y accesible desde un archivo principal index.html, compartiendo un diseño coherente y unificado mediante un archivo CSS global.

## Estructura del Repositorio

Un archivo principal index.html como página de inicio.

Cada actividad estará en su propia carpeta:

/actividad02
/actividad04
/actividad06
/actividad08
/actividad10
/actividad12
/actividad14


Todas las actividades deben poder abrirse desde enlaces o botones en index.html.
